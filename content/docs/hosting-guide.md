## 空间上传教程

本教程介绍如何在 free-host.cwxian.com 上传或同步网站文件，让你的项目快速上线。

### 支持的网站类型

免费托管空间主要支持：

- 静态网站（HTML、CSS、JavaScript）
- 个人/项目展示页
- 有限动态脚本（具体支持范围以子站说明为准）

### 通过 Web 界面上传

1. 登录 [free-host.cwxian.com](https://free-host.cwxian.com)；
2. 进入文件管理器；
3. 选择或拖拽网站文件到上传区域；
4. 等待上传完成，访问分配的域名查看效果。

### 通过 Git 同步

如果你的项目代码托管在 Git 仓库中，可以使用 Git 同步功能：

1. 在子站绑定你的 Git 仓库地址；
2. 配置自动部署或手动触发同步；
3. 每次推送代码后，子站会自动拉取最新文件。

### 目录结构建议

```
/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── app.js
└── assets/
    └── logo.png
```

确保站点根目录包含 `index.html`，这是默认访问入口。

### 常见问题

#### 上传后页面空白

1. 检查 `index.html` 是否位于根目录；
2. 检查资源引用路径是否正确；
3. 查看浏览器控制台是否有报错。

#### 空间超出限制

如果网站文件超过分配的存储空间，建议：

1. 压缩图片和静态资源；
2. 移除不必要的文件；
3. 优化网站体积后重新上传。
