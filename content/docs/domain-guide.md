## 域名解析教程

本教程介绍如何在 dns.cwxian.com 管理你的二级域名解析记录，以及常见 DNS 问题的排查方法。

### 登录子站

审核通过后，使用申请时填写的邮箱或账号登录 [dns.cwxian.com](https://dns.cwxian.com)。

### 添加解析记录

进入域名管理页面，你可以为域名添加以下类型的解析记录：

#### A 记录

将域名指向一个 IPv4 地址，常用于指向服务器或托管空间。

- **主机记录**：通常为 `@` 或 `www`
- **记录值**：你的服务器 IP 地址

#### CNAME 记录

将域名指向另一个域名，常用于指向 CDN 或托管空间提供的域名。

- **主机记录**：通常为 `@` 或 `www`
- **记录值**：目标域名，如 `your-project.github.io`

#### TXT 记录

用于域名验证、SPF、DKIM 等场景。

### 解析生效时间

DNS 解析记录修改后，通常需要几分钟到 48 小时在全球范围内生效。你可以通过以下方式检查：

```bash
nslookup yourname.cwxian.com
```

### 常见问题

#### 域名无法访问

1. 检查解析记录是否正确；
2. 确认目标服务器或空间是否正常运行；
3. 等待 DNS 缓存刷新。

#### 解析记录不生效

1. 检查 TTL 设置，较短的 TTL 会更快生效；
2. 使用 `dig` 或 `nslookup` 查询当前解析结果；
3. 清除本地 DNS 缓存后重试。
